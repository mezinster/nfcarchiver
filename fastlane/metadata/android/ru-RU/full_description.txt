Распределённая система архивации данных с использованием NFC-меток. Храните файлы на нескольких NFC-метках и восстанавливайте их позже.

Разбейте любой файл на части и запишите их на NFC-метки. Сканируйте метки в произвольном порядке для сборки и восстановления исходного файла.

Возможности:
* Архивация файлов — разбиение любого файла на части для записи на NFC-метки
* Архивация текста — хранение текстовых сообщений на NFC-метках
* Восстановление — сканирование меток в произвольном порядке и сборка исходного файла
* Сжатие — опциональное GZIP сжатие для уменьшения количества меток
* Шифрование — AES-256-GCM шифрование с паролем (PBKDF2 для ключа)
* Офлайн работа — не требует подключения к сети

Поддерживаемые NFC-метки:
* NTAG213 (144 байт)
* NTAG215 (504 байт)
* NTAG216 (888 байт)
* MIFARE Ultralight (48 байт)
* MIFARE Ultralight C (144 байт)

Использует бинарный формат NFAR v1 с 28-байтовым заголовком, проверкой целостности CRC32 и идентификацией архива по UUID. Чанки можно сканировать в любом порядке, проверка происходит автоматически.

NFC Архиватор — свободное программное обеспечение с открытым исходным кодом. Без отслеживания, без рекламы, без доступа к сети.
